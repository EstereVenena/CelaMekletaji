# 🌐 Ceļa Meklētāji

Mūsdienīga web platforma jauniešu organizācijas darbības atbalstam – informācijas publicēšanai, klubu pārvaldībai un lietotāju mijiedarbībai.

---

## 📌 Par projektu

**Ceļa Meklētāji** ir tīmekļa sistēma, kas izstrādāta, lai:
- nodrošinātu centralizētu informācijas pieejamību
- attēlotu klubus interaktīvā kartē
- atvieglotu lietotāju reģistrāciju un pārvaldību
- uzlabotu komunikāciju starp organizāciju un dalībniekiem

Projekts izstrādāts kā kvalifikācijas darbs.

---

## ⚙️ Izmantotās tehnoloģijas

- **Frontend:**
  - HTML5
  - CSS3 (custom responsive design)
  - JavaScript
  - Leaflet.js (kartes funkcionalitātei)

- **Backend:**
  - PHP
  - MySQL

- **Citi rīki:**
  - Git & GitHub (versionēšana)
  - Font Awesome (ikonas)

---

## 🚀 Galvenā funkcionalitāte

- 🌍 Interaktīva karte ar klubu attēlojumu
- 🧾 Lietotāju reģistrācija un autentifikācija
- 📊 Administrācijas panelis
- 🖼️ Galerija ar lightbox funkcionalitāti
- 📱 Responsīvs dizains (desktop + mobile)
- 🧩 Modulāra struktūra (header/footer, reusable komponentes)
