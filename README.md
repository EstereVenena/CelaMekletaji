# 🌐 Ceļa Meklētāji

Tīmekļa vietne biedrības **“Ceļa meklētāji”** darbības atbalstam.
Projekts izstrādāts kā kvalifikācijas darbs.

---

## 🔗 Mājaslapas atvēršana

Mājaslapu var atvērt pārlūkprogrammā, ievadot projekta adresi:

```text
https://kristovskis.lv/4pt/venena/celamekletaji/
```
---

## 📌 Par projektu

**Ceļa Meklētāji** ir tīmekļa platforma, kas paredzēta jauniešu organizācijas informācijas publicēšanai, klubu attēlošanai, lietotāju reģistrācijai un pieteikumu pārvaldībai.

Sistēmas mērķis ir nodrošināt vienotu vietu, kur lietotāji var apskatīt jaunumus, klubus, galeriju un pieteikties aktivitātēm.

## ⚙️ Izmantotās tehnoloģijas

* HTML5
* CSS3
* JavaScript
* PHP
* MySQL
* Leaflet.js
* Font Awesome
* Git / GitHub

---

## 🚀 Galvenā funkcionalitāte

* Lietotāju reģistrācija un pieslēgšanās
* Lietotāju lomu sistēma
* Administrācijas panelis
* Klubu pārvaldība
* Interaktīva karte ar klubiem
* Jaunumu publicēšana
* Pasākumu un nodarbību pārvaldība
* Galerija ar attēlu apskati
* Vecāka un skolēna paneļi
* Responsīvs dizains mobilajām ierīcēm

---

## 👥 Lietotāju lomas

Sistēmā paredzētas vairākas lietotāju lomas:

* Apmeklētājs
* Skolēns
* Vecāks
* Skolotājs
* Direktors
* Administrators

Katrai lomai ir atšķirīgas piekļuves tiesības un pieejamās funkcijas.
